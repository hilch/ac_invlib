Configuration of an ACOPOSinverter P86 connected to Powerlink (EPL)

device's name can be read from the physical view:

Open the I/O - Configuration and add 'HMIS', 'LFT', 'OTR', 'LCR', 'THR' and 'THD' to 'Optional monitoring data points'

and set 'Download ACPi parameter set' to 'no':

Set 'Function model' to 'Direct control' and 'Download initial values' to 'no'

Open the I/O - Mapping and connect the 'io_mapping'- elements
